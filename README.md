If you want to launch it via windows use [this](https://github.com/maowdic/RepoStylist/releases/tag/V-1.0)

However if you want to launch it from linux follow these steps:
```
apt update && apt upgrade

apt install python git

git clone https://github.com/maowdic/RepoStylist.git

cd RepoStylist

pip install -r REQUIREMENTS

python main.py
```
